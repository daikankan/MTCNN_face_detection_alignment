# MTCNN_face_detection_alignment
Tensorflow version of "Joint Face Detection and Alignment using Multi-task Cascaded Convolutional Neural Networks"
## Requirement
1. Python 3.5
2. Tensorflow_1.10.0 or later
3. Cuda (if use nvidia gpu)
